Step 1 - Install WSL2 with Debian

References

WSL Install instructions are at: * https://learn.microsoft.com/en-us/windows/wsl/install - Install instructions for Win10. * https://learn.microsoft.com/en-us/windows/wsl/install-on-server - Install Instructions for Win2022.

To list avaialable Linux distributions run:

> wsl --list --online

The following is a list of valid distributions that can be installed.
Install using 'wsl --install -d <Distro>'.

NAME            FRIENDLY NAME
Ubuntu          Ubuntu
Debian          Debian GNU/Linux
kali-linux      Kali Linux Rolling
openSUSE-42     openSUSE Leap 42
SLES-12         SUSE Linux Enterprise Server v12
Ubuntu-16.04    Ubuntu 16.04 LTS
Ubuntu-18.04    Ubuntu 18.04 LTS
Ubuntu-20.04    Ubuntu 20.04 LTS

Connect via RDP:

mstsc /admin /v:{automationworkerwindowsnode}

Login as user {automationworkerwindowsuseradministrator} and open a command prompt.

Execute the following script: 

                
$WIN10_STRING = "Windows 10"
$WIN2016_STRING = "Windows Server 2016"
$WIN2019_STRING = "Windows Server 2019"
$WIN2022_STRING = "Windows Server 2022"

$ERROR_STRING = "Please use Windows Desktop 10 or Windows Server 2022 for WSL2."

$LINUX_DISTRIBUTION = "Debian"

if ((systeminfo | select-string -pattern $WIN10_STRING)) {
    # Win10
    Write-Host "On ${WIN10_STRING}"
    wsl --install -d "${LINUX_DISTRIBUTION}"
    
} elseif ((systeminfo | select-string -pattern $WIN2016_STRING)) {
    # Win2016
    Write-Host "On ${WIN2016_STRING}"
    Write-Host "$ERROR_STRING"
    exit 1
    
} elseif ((systeminfo | select-string -pattern $WIN2019_STRING)) {
    # Win2019
    Write-Host "On ${WIN2019_STRING}"
    Write-Host "$ERROR_STRING"
    exit 1

} elseif ((systeminfo | select-string -pattern $WIN2022_STRING)) {
    # Win2022
    Write-Host "On ${WIN2022_STRING}"
    wsl --install -d "${LINUX_DISTRIBUTION}"
}

Step 3 - Verify WSL is Installed

Execute the following script: 

                
wsl --list

if (echo $?) {
    Write-Host "WSL is installed."
} else {
    Write-Host "WSL is not installed."
    exit 1
}
